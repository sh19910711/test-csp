## Usage

1. Run web server
  - $ bundle exec rackup
2. Access to app from browser extensions
3. Check by eye

## URL

* `/some/api`
  - it returns simple json text
* `/api/with/custom/header`
  - it also returns simple json text and custom header (X-Hello)
